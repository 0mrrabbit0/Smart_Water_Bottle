# OLED Display Simulator

这是一个用于预览智能水杯项目 OLED 显示效果的 Web 模拟器。

## 功能特性

- ✅ 模拟 SSD1306 128x64 单色 OLED 屏幕
- ✅ 实时显示传感器数据（温度、TDS、水位、电池电量）
- ✅ 5 种显示模式切换
- ✅ 自动更新时间显示
- ✅ 响应式界面设计

## 5 种显示模式

1. **正常模式** - 显示所有传感器数据和电池电量
2. **解锁数字** - 密码解锁时的数字滚动界面
3. **解锁提示** - 密码输入步骤提示
4. **报警模式** - 3 次解锁失败后的警报界面
5. **锁定模式** - 锁定状态下的简化显示

## 快速开始

### 1. 安装依赖

```bash
pip install flask
```

或者使用 pip3：

```bash
pip3 install flask
```

### 2. 运行模拟器

在项目根目录下运行：

```bash
python oled_simulator.py
```

或者：

```bash
python3 oled_simulator.py
```

### 3. 打开浏览器

服务器启动后，在浏览器中访问：

```
http://localhost:5000
```

或者从局域网其他设备访问（查看终端显示的 IP 地址）：

```
http://<your-ip>:5000
```

## 界面说明

### OLED 屏幕区域
- 中央黑色区域模拟实际的 OLED 显示屏
- 分辨率：128x64 像素
- 显示比例：4倍放大便于查看

### 模式切换按钮
点击不同的按钮可以切换显示模式，实时查看不同场景下的界面效果。

### 信息面板
显示当前显示器的技术规格和当前模式。

## 电池图标设计

所有显示页面的右上角都会显示动态电池图标：

- **位置**: 右上角 (110, 0)
- **尺寸**: 16x10 像素（包含文字）
- **组成**:
  - 外框：14x8 像素矩形
  - 正极：2x4 像素凸起
  - 内部填充：根据电量百分比动态填充
  - 百分比数字：显示在电池图标下方
- **动态效果**:
  - 100% - 满电（完全填充）
  - 50% - 半满（填充一半）
  - 0% - 空电（无填充）

电池图标示例：
```
85% 电量:          20% 电量:          0% 电量:
┌──────────┐─      ┌──────────┐─      ┌──────────┐─
│▓▓▓▓▓▓▓▓  │ │     │▓▓        │ │     │          │ │
└──────────┘─      └──────────┘─      └──────────┘─
    85%               20%               0%
```

## 技术细节

- **后端**: Flask (Python)
- **前端**: HTML5 Canvas + JavaScript
- **刷新率**: 200ms（模拟实际OLED刷新频率）
- **时间更新**: 1秒/次
- **字体**: Courier New 等宽字体（模拟像素字体）
- **电池图标**: Canvas绘制，动态填充效果

## 注意事项

1. 模拟器使用等宽字体近似显示，实际OLED使用6x8和8x16点阵字体
2. 显示位置可能与实际硬件略有差异，仅供预览参考
3. 电池电量、温度等数据为模拟值，可修改 `oled_simulator.py` 中的 `sensor_data` 字典进行调整

## 自定义数据

编辑 `oled_simulator.py` 中的 `sensor_data` 字典：

```python
sensor_data = {
    'temperature': 25.3,      # 温度（°C）
    'tds_value': 120,         # TDS值（ppm）
    'water_level': 2,         # 水位 0=空 1=低 2=中 3=满
    'battery_capacity': 85,   # 电池电量（0-100%）
    'time': '12:34:56',       # 时间（自动更新）
    'display_mode': 'normal'  # 当前模式
}
```

## 停止服务器

在终端按 `Ctrl + C` 停止 Flask 服务器。

## 故障排除

### 端口被占用
如果 5000 端口已被占用，修改 `oled_simulator.py` 最后一行：

```python
app.run(debug=True, host='0.0.0.0', port=5001)  # 改为其他端口
```

### Flask 未安装
确保已安装 Flask：

```bash
pip install flask
# 或
pip3 install flask
```

### 浏览器无法访问
检查防火墙设置，确保允许访问 5000 端口。
